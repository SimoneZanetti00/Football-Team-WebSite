# Deportivo Lo Stanzino — Team Website ⚽

This repository contains the early development version of the official website for **Deportivo Lo Stanzino**, a small football team project created by friends.

The website is currently **a work in progress** and serves as an experimental platform to build a simple but modern online presence for the team.

---

## Project Status

🚧 **Development Stage**

This website is still in an early development phase.  
Some elements are temporary and will be improved over time, including:

- layout refinements
- visual design
- image optimization
- additional team content
- shop presentation

The goal is to gradually evolve this prototype into a clean and modern team website.

---

## Current Features

At the moment the website includes:

- **Home page**
- **Team page** with player photos
- **Merch section** with a few team items
- **Basic responsive layout**
- **Custom styling and simple animations**

This version focuses mainly on structure and content placement rather than final design.
